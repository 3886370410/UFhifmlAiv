# 在线考试管理系统 java212

## 项目概述

在线考试管理系统是一款基于SpringBoot框架开发的应用，采用前后端分离的架构。系统支持多种题型，包括单选题、多选题、判断题、填空题和主观题等，为教育行业提供便捷的在线评测工具。

## 技术栈

- 后端：SpringBoot、Mybatis-Plus、JPA
- 数据库：MySQL
- 前端：Vue、ElementUI

## 功能模块

### 系统角色

#### 管理员（Admin）

- 用户管理：管理员、教师、学生
- 考试管理：新增、删除考试，分配试题，查看考试详情，评分，主观题评分，查看分数
- 考试通知管理
- 班级成绩管理
- 题目分析管理

#### 教师（Teacher）

- 学生信息查询
- 考试管理：新增、删除考试，分配试题，查看考试详情，评分，主观题评分，查看分数
- 考试通知管理
- 班级成绩管理
- 题目分析管理

#### 学生（Student）

- 参加考试
- 查看分数
- 查看考试详情
- 查看考试通知
- 查看班级成绩
- 查看题目分析

## 运行环境

- 操作系统：Win10/11
- 开发工具：ideaIU-2022.3.3.win
- 开发语言：Java (jdk-8u371)、Vue.js(node.js 13.14)
- 项目管理工具：apache-maven-3.9.4
- 数据库：MySQL 8.0

## 部署步骤

1. 执行目录中的sql文件
2. 解压项目压缩包
3. 使用IDEA导入项目
4. 等待Maven下载相关依赖，完成后点击后端项目启动
5. 使用npm启动前端项目

## 目录结构

```
├── backend                 // 后端代码目录
├── frontend                // 前端代码目录
└── sql                     // 数据库脚本目录
    └── init.sql            // 初始化数据库脚本
```
